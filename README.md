# 🍽️ KRMU Bites – Smart Canteen Website

## 🌐 Live Demo

**Website:** https://krmu-test-canteen.vercel.app/

## 📖 Overview

KRMU Bites is a modern and responsive canteen website designed for students and visitors of K.R. Mangalam University. The platform provides an attractive user interface for exploring food items, offers, and canteen-related information.

The website focuses on delivering a smooth user experience with a clean design, responsive layout, and engaging visuals.

---

## ✨ Features

* Responsive design for desktop, tablet, and mobile devices
* Modern and attractive UI
* Interactive navigation
* Food showcase section
* Smooth user experience
* Optimized layout and styling
* Easy-to-maintain code structure

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/KumarSubodh12/krmu-test-canteen.git
```

### Navigate to Project

```bash
cd krmu-test-canteen
```

### Run Locally

Open `index.html` directly in your browser

or use VS Code Live Server:

```bash
Right Click → Open with Live Server
```
